# <a name="linux-virtual-machines-documentation"></a>Linux 虚拟机文档

Azure Linux 虚拟机可以使用 Red Hat、Ubuntu 或你选择的 Linux 发行版，按需提供具有高可伸缩性，并且十分安全的虚拟化基础结构。 使用我们的快速入门、教程和示例了解如何创建、配置、管理和缩放 Linux VM。
