启用系统分配的托管标识只需单击一次即可。 可以在创建 VM 的过程中或在现有 VM 的属性中启用它。

![新建存储帐户](./media/active-directory-msi-tut-enable/identity.png)


**若要在新 VM 上启用系统分配的托管标识，请执行以下操作：** 

1. [登录到 Azure 门户](https://portal.azure.com)

2. [创建启用了系统分配标识的虚拟机](/azure/active-directory/managed-identities-azure-resources/qs-configure-portal-windows-vm#system-assigned-managed-identity)
