#### <a name="to-get-the-storsimple-service-registration-key"></a>获取 StorSimple 服务注册密钥

1.  在“StorSimple 设备管理器”边栏选项卡上，单击你创建的服务。 这会在右侧打开一个新的边栏选项卡。

2.  在打开的边栏选项卡中，单击“管理” “密钥”。**&gt;**

3.  单击复制图标以复制服务注册密钥并将其保存供日后使用。
