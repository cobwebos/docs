## <a name="prerequisites"></a>必备组件 
要完成本教程，必须将一个 VHD 文件上传到与要在其中创建自定义映像的实验室相关联的存储帐户。 选择以下任一链接上传 VHD 文件：

- [使用 AzCopy 命令行实用工具上传](../articles/lab-services/devtest-lab-upload-vhd-using-azcopy.md)
- [使用 Microsoft Azure 存储资源管理器上传](../articles/lab-services/devtest-lab-upload-vhd-using-storage-explorer.md)
- [使用 PowerShell 控制台上传](../articles/lab-services/devtest-lab-upload-vhd-using-powershell.md)
