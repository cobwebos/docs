> [!IMPORTANT]
> [Microsoft 标识平台（v2.0）](/azure/active-directory/develop/v2-overview)是 Azure Active Directory （Azure AD）开发人员平台（1.0 版）的演变。 开发人员可以通过它来生成应用程序，从而可以采用所有 Microsoft 标识登录，以及获取令牌来调用 Microsoft Graph 等 Microsoft API 或开发人员生成的 API。
> 此内容适用于较旧的 Azure AD 1.0 版终结点。 建议对新项目使用 v2.0 终结点。 有关详细信息，请阅读[为什么要更新 Microsoft 标识平台（v2.0）？](/azure/active-directory/azuread-dev/azure-ad-endpoint-comparison) 以及[Microsoft 标识平台的限制](/azure/active-directory/azuread-dev/azure-ad-endpoint-comparison#limitations)。
