# <a name="orders-dashboard-in-commercial-marketplace-analytics"></a>商业 Marketplace 分析中的订单仪表板

本文提供了有关合作伙伴中心中的 "**订单" 仪表板**的信息。 此仪表板以图形和可下载的格式显示有关订单的信息。

若要在合作伙伴中心分析工具中访问 "**订单" 仪表板**，请在 "商业市场" 下打开 " **[分析" 仪表板](https://partner.microsoft.com/dashboard/commercial-marketplace/analytics/summary)** 。

>[!NOTE]
> 有关分析术语的详细定义，请参阅[商业 Marketplace 分析的常见问题和术语](./faq-terminology.md)。

## <a name="orders-dashboard"></a>订单仪表板

"**分析**" 菜单的 "**订单" 仪表板**显示所有 SaaS 产品/服务的当前订单。 您可以查看以下各项的图形表示形式：

- [订单摘要](#order-summary)
- [按地理位置排序](#orders-by-geography)
- [按产品/服务排序](#orders-by-offers)
- [每个站点与每个客户的订单趋势](#orders-trend-per-site-versus-per-seat)
- [按 Sku 排序](#orders-by-skus)
- [订单和座位趋势](#orders-and-seats-trend)
- [订单详细信息表](#order-details-table)

> [!NOTE]
> 分析报表在云合作伙伴门户（CPP）中的显示方式与合作伙伴中心内新的商用 Marketplace 计划之间存在差异。 一个具体的方法是，CPP 中的**卖方 Insights**具有**订单 & 使用量**"选项卡，该选项卡显示基于使用情况的产品/服务和基于使用情况的产品/服务的数据。 在合作伙伴中心，"**订单**" 页上有一个用于 SaaS 产品的单独选项卡。

## <a name="order-dashboard-details"></a>订单仪表板详细信息

本部分更详细地介绍分析报表。

### <a name="order-summary"></a>订单摘要

"订单摘要" 部分显示所有采购订单（未包括已取消订单）、已取消订单和座位的计数。

"总订单" 旁边的百分比值表示所选日期范围的增长量。

![合作伙伴中心分析订单摘要](./media/order-summary.png)

- 向上的绿色三角形表示正增长趋势。
- 向下的红色三角形表示相对于上个月的负增长趋势。
- 增长趋势由微条形图表示。 您可以通过将鼠标悬停在图表中的列上来显示每个月的值。
- "已取消" 订单是以前在所选日期范围内购买并取消的订单计数。
- 座位是在所选日期范围内创建的座位数。

### <a name="orders-by-geography"></a>按地理位置排序

"**按地理位置排序**" 热度地图显示世界地图上的订单计数，并显示基于客户国家/地区的地图。 此热度地图的工作方式与 **[按地域热度地图的客户](./customer-dashboard.md#customer-by-geography)** 的功能相同。

![合作伙伴中心按地理位置分析订单](./media/orders-by-geography.png)

### <a name="orders-by-offers"></a>按产品/服务排序

**按顺序提供**的 "环形图" 根据产品的产品/服务名称组织订单（包括已取消的订单）。

- 顶部的产品/服务将显示在关系图中，其余的产品/服务将按 "全部 Rest" 分组。
- 可在图例中选择特定产品/服务，以便仅在图形中显示这些产品/服务。
- 将鼠标悬停在图中的某一切片上将显示与所有产品/服务的订单总数相比，该产品/服务的订单数和百分比。
- "**按产品提供的订单" 趋势**显示每月增长趋势。 月份列按产品/服务名称表示订单数。 折线图显示在 z 轴上绘制的增长百分比趋势。
- 您可以使用图表顶部的滑块沿 x 轴向右和向左滚动，并专注于特定的数据点。
- 您可以通过选择图例上的特定项来显示趋势图表。
- 你还可以选择显示**已取消订单**的趋势和数据。 图形的工作方式与 "**按产品提供订单**" 图形的工作方式相同。

### <a name="orders-trend-per-site-versus-per-seat"></a>每个站点与每个客户的订单趋势

**每个站点和每个座位**环形图表示客户购买的每个站点 saas 和每个客户的 saas 订单（此图表包含已取消的订单）的细分。 柱形图表示客户购买的每个站点 SaaS 和每个客户 SaaS 订单的趋势（此图表包含已取消的订单）。

### <a name="orders-by-skus"></a>按 Sku 排序

"**按 Sku 排序**" 图表代表所有产品/服务的库存单位（SKU）级别的订单趋势，这包括已取消的订单。 环形图表示前五个 SKU 订单的明细，柱形图表示前5个 Sku 的订单趋势。

### <a name="orders-and-seats-trend"></a>订单和座位趋势

"**订单" 和 "座位" 趋势**图显示最高的50产品和订单数。 它们显示在领导板上，并按最高订单计数和订单百分比排序。

- **按 sku**列出的订单：选择产品/服务可查看图表中前五个 sku 的订单计数细目。
- **由 sku**提供的座位：前5个 sku 的每月座位趋势。 如果你选择的产品/服务不是每个客户的产品/服务，你将在此面积图中看不到任何数据。

### <a name="canceled-orders-by-offers"></a>按产品/服务取消订单

"**已取消的订单" 产品/服务**饼图根据您的产品/服务名称组织您的所有已取消的订单。 最上面的产品/服务将显示在图中，其余的产品/服务将按 "全部 Rest" 分组。 可在图例中选择要在图表中显示的特定产品/服务。

- 将鼠标悬停在图中的某一切片上将显示与所有产品/服务的订单总数相比所选产品的订单数和百分比。
- 柱形图显示月趋势。 列按产品名称表示已取消的订单数。 您可以使用图表顶部的滑块沿 x 轴向右和向左滚动，并专注于特定的数据点。 您可以通过选择图例上的特定项来显示趋势图表。

### <a name="order-details-table"></a>订单详细信息表

"订单详细信息" 表显示按购置日期排序的1000顶部订单的编号列表。

- 网格中的每一列都是可排序的。
- 如果记录计数小于1000，则可以将数据提取到 TSV 文件中。
- 如果记录数超过1000，则导出的数据将在接下来的30天内异步置于下载页中。
- 筛选器可应用于 "**订单详细信息" 表**，以仅显示感兴趣的数据。 可以按国家/地区、Azure 许可证类型、Marketplace 许可证类型、产品/服务类型、订单状态、免费跟踪、Marketplace 订阅 ID、客户 ID 和公司名称对数据进行筛选。

#### <a name="orders-page-filters"></a>订单页筛选器

在页面级别应用这些筛选器。

您可以选择多个筛选器来呈现您选择的条件的图表，以及要在**详细订单数据**网格/导出中显示的数据。 筛选器将应用于在 "订单" 页右上角为所选数据范围提取的数据。

- 仅在所选日期范围内提供订单的产品/服务类型和产品名称。 列表中的产品/服务名称将显示在列表中选择的产品/服务类型。
- 应用的筛选器显示每个选定筛选器的每个选择范围内的总度量值。 选择默认选定内容时，不会显示应用的筛选器。
- 如果为其中一个下拉列表选择 "**全部**"，则将聚合所选页面中的所有度量值。 例如： "产品/服务类型" 筛选器选项中的 "全部" 表示已选择所有产品/服务类型。 这是下拉列表的默认选择。 选定的筛选器将不会**显示任何内容**。
- **多值选择**：将为下拉列表下所做的所有选择聚合页中的所有度量值。 如果进行了多个选择，应用的筛选器将显示所做的所有选择的计数。 有关参考，请参阅下图。

    ![合作伙伴中心分析顺序，其中包含应用于筛选器的多个值](./media/filters-applied.png)

- **单值选择**：如果选择一个值，应用的筛选器将显示选定的一个筛选器的计数。 请参阅下图以了解参考。

     ![合作伙伴中心分析顺序，其中单个值应用于筛选器](./media/filters-applied-single.png)
