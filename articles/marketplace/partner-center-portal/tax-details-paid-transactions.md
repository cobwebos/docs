# <a name="tax-details-for-commercial-marketplace-publishers"></a>商业市场发布者的税务详细信息

若要了解有关价格选择的税务注意事项，以及 [Microsoft Azure 市场发行者协议](https://go.microsoft.com/fwlink/p/?LinkID=699560) 或新式出版商协议下的税务责任，请参阅下表中的国家/地区。

商业扩张调整可能会在有限的几个国家/地区规定实际付款。 有关商业扩张调整及其适用的市场的详细信息，请参阅 [Microsoft Azure 市场发布者协议](https://go.microsoft.com/fwlink/p/?LinkID=699560)和受影响国家/地区的列表。
