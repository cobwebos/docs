# <a name="azure-security-center-for-iot-security-alerts"></a>用于 IoT 安全警报的 Azure 安全中心

用于 IoT 的 Azure 安全中心使用高级分析和威胁智能来持续分析 IoT 解决方案，以提醒你应对恶意活动。
此外，你可以根据预期设备行为的了解来创建自定义警报。
警报可作为潜在危害的指标，并应进行调查和修正。

在本文中，你将找到内置警报列表，这些警报可在 IoT 中心和/或 IoT 设备上触发。
除了内置警报外，Azure 安全中心 IoT 还允许你根据预期的 IoT 中心和/或设备行为定义自定义警报。
有关更多详细信息，请参阅[可自定义警报](concept-customizable-security-alerts.md)。
