# <a name="azure-maps-routing-coverage"></a>Azure Maps 路由覆盖区域

本文提供了有关 Azure Maps 路由的覆盖区域信息。 在搜索最佳位置 A 到 B 的路线时，Azure Maps 可以提供非常准确的行程时间，实时更新后的旅行信息和路由说明。 采用的路由将帐户当前流量、 车辆类型和条件等的其他搜索词以避免。 因为 Azure Maps 具有各种级别的信息和不同的区域的准确性，优化路由的能力取决于区域。 下表列出的区域以及可以为其请求的信息种类。
 
查看[**地理编码**](geocoding-coverage.md)的覆盖区域。       
查看[**流量**](traffic-coverage.md)的覆盖区域。  
查看[呈现](render-coverage.md)的覆盖区域。
<br>  
**图例**

| 符号             | 含义                                |
|--------------------|----------------------------------------|
| ✓                  | 国家/地区获得详细数据。   |
| Ø                  | 国家/地区提供简化的数据。 |
| 缺少国家/地区 | 未提供国家/地区数据。          |
