# <a name="keyboard-shortcuts-in-the-azure-portal"></a>Azure 门户中的键盘快捷方式

本文列出了在 Azure 门户中的任意位置工作的键盘快捷键。 各个服务可能有其自己的键盘快捷方式。

## <a name="actions"></a>操作

|若要此操作 |按 |
| --- | --- |
|创建资源|G+N|
|打开**所有服务**|G+B|
|搜索资源、服务和文档|G+/|
|搜索资源菜单项|CTRL+/ |
|所选的左侧的边栏项上移 |ALT+Shift+向上键|
|向下移动所选的左侧的边栏项 |ALT+Shift+向下键|

## <a name="navigation"></a>导航

|若要此导航 |按 |
| --- | --- |
|将焦点移到命令栏 |G+, |
|标头和左侧的边栏之间切换焦点 | G+. |

## <a name="go-to"></a>转到

|若要转到此位置 |按 |
| --- | --- |
|转到**仪表板** |G+D |
|转到**的所有资源**|G+A |
|转到**资源组**|G+R |
|打开位于此位置的左侧的边栏项 |G+数字|
