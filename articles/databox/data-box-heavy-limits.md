## <a name="data-box-heavy-limits"></a>Data Box Heavy 限制

- Data Box Heavy 最多可为每个节点存储1000000000个文件。
- Data Box Heavy 在云中最多支持512个容器或共享。 用户共享中的顶层目录成为云中的容器或 Azure 文件共享。
