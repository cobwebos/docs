# <a name="build-real-time-applications-with-azure-signalr-service"></a>通过 Azure SignalR 服务构建实时应用程序

借助[Azure SignalR 服务](https://azure.microsoft.com/services/signalr-service/)，可以轻松地将实时功能添加到应用程序。 此实时功能允许服务将内容更新推送到连接的移动应用程序。 因此，将更新应用，而无需轮询服务器或提交新的更新 HTTP 请求。

## <a name="azure-signalr-service-features"></a>Azure SignalR 服务功能
- 广泛的客户端支持，包括 web 和移动浏览器、桌面应用程序、服务器进程、IoT 设备和游戏控制台等各种客户端。
- 允许多个实例一起使用，以扩展到数百万个移动应用程序连接。 它还支持多个全球区域，以实现分片、高可用性或灾难恢复目的。
- 为不同消息模式提供丰富的 Api，以将消息发送到特定连接、所有连接或属于特定用户或被放入任意组的连接子集。
