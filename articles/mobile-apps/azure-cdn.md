# <a name="content-delivery-network-on-azure-with-azure-content-delivery-network"></a>Azure 上的内容交付网络与 Azure 内容交付网络
[Azure 内容分发网络](https://azure.microsoft.com/services/cdn/)为开发人员提供了一个全局解决方案，通过在世界各地将视频、图像或静态文件的内容缓存到数百个点来向用户快速交付高带宽内容，例如视频、图像或静态文件。 内容交付网络还可以加速动态内容（无法缓存这些内容）以减少加载时间、节省带宽和提高响应能力。 无论你是开发还是管理网站或移动应用程序，或者编码和分发流媒体、游戏软件、固件更新或 IoT 终结点，你都可以获得这些优势。

## <a name="azure-content-delivery-network-features"></a>Azure 内容分发网络功能
  - 与 Azure 服务（如 web 应用程序、媒体服务、存储和云服务）无缝协作。
  - 轻松处理流量峰值和繁重负载。
  - 与 Akamai 和 Verizon 集成，以构建和定制内容交付策略。
  - 使用动态站点加速。
  - 控制内容分发网络缓存行为与缓存规则。
  - 获取 HTTPS 自定义域支持。
  - 查看 Azure 诊断日志可查看核心分析并将其导出到 Azure Blob 存储、Azure 事件中心或 Azure Monitor。
  - 通过压缩文件来提高性能。
  - 使用地理位置筛选按国家或地区限制内容交付网络内容。
