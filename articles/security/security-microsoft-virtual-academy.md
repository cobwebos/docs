# <a name="azure-security-courses-from-microsoft-virtual-academy"></a>Microsoft Virtual Academy 中的 Azure 安全性课程
Microsoft Virtual Academy 提供免费在线培训，帮助开发人员、IT 和数据专业人员以及学生了解最新技术、构建自己的技能并提升他们的职业生涯。

在此页上，你会发现特选的与 Azure 安全性相关的课程集合。 若要查看所有可选课程，请访问 [Microsoft Virtual Academy](https://mva.microsoft.com/)。

[云中的开发/测试](https://mva.microsoft.com/en-us/training-courses/devtest-in-the-cloud-16274?l=9aAijd7LC_2005190311)

你是否是需要提供更快更好的应用程序的开发人员？ 将开发和测试移到云环境，可帮助你精准实现此目标！ 了解如何开展工作，并找出进行此项移动的好处。 此外，请参阅演示及演示文稿，了解 Microsoft Azure 如何支持你的开发和测试需求。 包含有关安全开发和部署实践的课程。

[Azure 上 Linux 的一般任务](https://mva.microsoft.com/en-us/training-courses/common-tasks-for-linux-on-azure-16191?l=J0Hvb7qJC_1204668937)

如果你有在 Microsoft Azure 平台上使用 Linux 的问题，此详细课程将为你解答。 与来自 [Linux Academy](https://linuxacademy.com/) 的专家一起探讨一些常见的任务。 了解如何在 Azure 中创建创建 Linux 虚拟机 (VM)、使用远程桌面软件访问 Linux VM，以及如何运行虚拟主机。 本课程介绍了许多安全技术和配置。

[保护云安全](https://mva.microsoft.com/en-us/training-courses/secure-the-cloud-14037?l=lQIkkst0B_5300115881)

在本课中，了解 Microsoft 如何使用新的安全控制帮助你符合全球合规性要求，如 ISO 27001 / 27018、FedRAMP、PCI 和 HIPAA。 这些控制的范围从待用数据加密、密钥管理、VM 保护、日志记录和监视，到反恶意软件服务、身份管理、访问控制等等。

[设计和实现云数据平台解决方案](https://mva.microsoft.com/en-us/training-courses/design-and-implement-cloud-data-platform-solutions-15711?l=jbCdW0j1B_3005244527)

了解 Microsoft 云数据平台解决方案的特性和功能 。 获取平台概述并了解相关安全功能、高可用性选项、监视和管理云数据的技术等等。 此外，获取有关指南，了解如何识别利弊，以及如何使用 Microsoft 云数据平台功能为公开和混合云解决方案作出决策。

[在云和移动环境中管理和保护身份](https://mva.microsoft.com/en-us/training-courses/manage-and-secure-identities-in-a-cloud-and-mobile-world-14013?l=GIJ2GcvrB_405192797)

在本课中，了解 Azure Active Directory 和 Microsoft 高级威胁分析如何帮助你保护并管理用户身份、在造成损害前找出安全漏洞，并且为你的用户提供单一身份用于访问公司的所有资源。 探索各种技术，用于发现 Shadow IT、管理应用程序访问，并通过高级安全报告、用户行为分析，审核并发出相应警报来监视可疑活动。

[启用云的环境中的安全](https://mva.microsoft.com/en-us/training-courses/security-in-a-cloudenabled-world-12725?l=CfLHobAcB_3904300474)

专家将引导你完成[企业架构师的 Microsoft 云安全](http://www.microsoft.com/download/48121)海报中的客户职责路线图。 专家还会提供有关现代化各部分安全状况的建议，包括监管、约束策略、安全操作、高价值的资产保护、信息保护和用户与设备的安全性，尤其着重保护管理控制。 了解 Microsoft 网络安全团队用来评估客户的云安全性并进行构建其安全性路线图的同一个框架。

[Microsoft Azure IaaS 深入探讨](https://mva.microsoft.com/en-us/training-courses/microsoft-azure-iaas-deep-dive-14339?l=PtppYVQgB_8300115888)

了解如何使用 Microsoft Azure 基础结构功能。 如果你是 IT 专业人员，无需具有以前使用过 Azure 的经验。 本课程将逐步指导你完成创建和配置 Azure 虚拟机、Azure 虚拟网络和跨界连接，以在云上启动和运行作业。 整个课程会包含安全功能和注意事项。

[开始适用于 IT 专业人员的 Azure 安全性](https://mva.microsoft.com/training-courses/getting-started-with-azure-security-for-the-it-professional-11165?l=HfHzCXSAB_7404300474)

本这个以演示为主的课程中，会由一组安全专家和 Azure 工程师带你通过基本认证并探索 Azure 中的可能性。 了解如何设计和使用各种技术，确保具有在云中启动项目所需的安全性和体系结构。 深入了解数据中心运营、VM 配置、网络体系结构和存储基础结构。

[深入了解 Azure Resource Manager 方案和模式](https://mva.microsoft.com/en-us/training-courses/deep-dive-into-azure-resource-manager-scenarios-and-patterns-13793?l=i1m06ZJYB_7001937557)

与专家团队一同探索 Azure Resource Manager，他们会向你展示脚本和工具，让你轻松启动或停止应用程序基础结的构元素。 探索如何使用基于角色的访问控制 (RBAC) ，以使用 Azure Resource Manager 实现安全性。

[Azure Rights Management Services 核心技能](https://mva.microsoft.com/en-us/training-courses/azure-rights-management-services-core-skills-10500?l=QLoxMwuCB_1805094681)

了解信息保护为什么是组织中的“必须”项目，以及权限管理如何保护组织的知识产权（只要通过设备和云）。 从 Microsoft 专家那里获取实践经验和技术专业知识。
