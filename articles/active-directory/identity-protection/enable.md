# <a name="enabling-azure-active-directory-identity-protection"></a>启用 Azure Active Directory Identity Protection

Azure Active Directory Identity Protection 是 Azure Active Directory (Azure AD) 的一项功能。 使用 Azure AD Identity Protection，你能够：

- 获得使用机器学习算法检测到的已标记用户和风险事件的综合视图
- 设置基于风险的条件访问策略以自动保护你的用户
- 通过对漏洞执行操作提高安全状态 

有关详细信息，请参阅 [Azure Active Directory Identity Protection](../active-directory-identityprotection.md)。

本文介绍如何启用 Azure Active Directory Identity Protection。

## <a name="steps-to-enable-azure-active-directory-identity-protection"></a>启用 Azure Active Directory Identity Protection 的步骤

1. 以全局管理员身份登录到 [Azure 门户](https://ms.portal.azure.com/) 

2. 在 Azure 门户中，单击“市场”。
   
    ![创建](./media/enable/01.png "创建")

3. 在应用程序列表中，单击“标识”。
   
    ![创建](./media/enable/05.png "创建")

4. 单击“Azure AD Identity Protection”。
   
    ![创建](./media/enable/06.png "创建")

5. 在“Azure AD Identity Protection”边栏选项卡中，单击“创建”。
   
    ![创建](./media/enable/04.png "创建")
