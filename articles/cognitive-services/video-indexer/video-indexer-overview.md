# <a name="what-is-video-indexer"></a>什么是视频索引器？

Azure 视频索引器是基于 Azure 媒体分析、Azure 搜索、认知服务（例如人脸 API、Microsoft Translator、计算机视觉 API 和自定义语音服务）生成的云应用程序。 借助视频索引器可以使用下面所述的视频索引器模型从视频中提取见解：
 
- **自动语言检测**：自动识别主导性的讲述语言。 支持的语言包括英语、西班牙语、法语、德语、意大利语、中文（简体）、日语、俄语和葡萄牙语（巴西）。 检测不到语言时会回退到英语。
- **音频听录**：将语音转换为 10 种语言的文本并允许扩展。 支持的语言包括英语、西班牙语、法语、德语、意大利语、中文（简体）、日语、阿拉伯语、俄语和葡萄牙语（巴西）。
- **隐藏式字幕**：以三种格式创建隐藏式字幕：VTT、TTML 和 SRT。
- **双通道处理**：自动检测、隔离脚本及合并到单个时间线。
- **噪声消减**：清理电话音频或有噪音的录制内容（基于 Skype 滤波器）。
- **脚本自定义 (CRI)**：训练和执行扩展的自定义语音转文本模型，以创建行业特定的脚本。
- **说话人枚举**：映射和了解哪个说话人在何时讲了哪些单词。
- **说话人统计信息**：提供说话人发言比率的统计数据。
- **视觉文本识别 (OCR)**：提取视频中显示的可视文本。
- **关键帧提取**：在视频中检测稳定的关键帧。
- **情绪分析**：在语音和视觉文本中识别积极、消极和中性情绪。
- **可视内容审核**：检测成人和/或猥亵的视觉内容。
- **关键字提取**：从语音和视觉文本中提取关键字。
- **标签识别**：识别显示的视觉对象和动作。
- **品牌提取**：从语音和视觉文本中提取品牌。
- **人脸检测**：检测和分组视频中显示的人脸。
- **人脸缩略图提取（“最佳人脸”）**：在每组人脸中自动识别捕获的最佳人脸（基于质量、大小和正面位置），并将其提取为图像资产。
- **名人识别**：基于 100 万个名人的数据库识别视频中的名人。 信息来源于 IMDB、维基百科和 Linkedin 中最有影响力的人物。
- **自定义人脸识别**：基于针对特定帐户训练的自定义模型识别视频中的人脸。
- **文本内容审核**：在音频脚本中检测确切的文本。
- **镜头检测**：确定视频中的场景何时发生了更改。
- **黑帧检测**：识别视频中的黑帧。
- **音效**：识别击掌、讲话和静音等音效。
- **主题推理**：从脚本中推理主要主题。 包括第一级 [IPTC](https://iptc.org/standards/media-topics/) 分类。
- **情感检测**：基于语音和音频提示识别情感。情感可能是：快乐、悲伤、愤怒或恐惧。
- **项目**：提取每个模型的丰富“下一种详细程度”项目。
- **翻译**：将音频脚本翻译成 54 种不同的语言。

等到视频索引器处理和分析完毕，你就可以查看、策展、搜索和发布视频见解。

不管你的角色是内容管理员还是开发人员，视频索引器服务都可以满足你的需求。 内容管理员可以通过视频索引器 Web 门户使用此服务，无需编写任何代码，详见[视频索引器网站入门](video-indexer-get-started.md)。 开发人员可以利用 API 来大规模处理内容，详见[使用视频索引器 REST API](video-indexer-use-apis.md)。 客户还可以通过此服务使用小组件来发布视频流以及在其自己的应用程序中提取的见解，详见[在应用程序中嵌入视觉小组件](video-indexer-embed-widgets.md)。

可以使用现有的 AAD、LinkedIn、Facebook、Google 或 MSA 帐户注册此服务。 有关详细信息，请参阅[入门](video-indexer-get-started.md)。

## <a name="scenarios"></a>方案

下面是一些可以使用视频索引器的方案

- 搜索 - 从视频中提取的见解可以用来增强整个视频库的搜索体验。 例如，对所说内容和人脸进行索引可以启用在视频中查找特定瞬间的搜索体验，视频中的特定的人可能说出了特定的内容，或者两个人出现在一起。 根据视频中的此类见解进行的搜索适用于新闻机构、教育机构、广播公司、娱乐内容所有者、企业 LOB 应用。推而广之，它适用于任何行业，前提是该行业有一个需要用户对其进行搜索的视频库。

- 盈利 - 视频索引器有助于提高视频的价值。 例如，依赖广告收入的行业（如新闻媒体、社交媒体等）可以利用提取出来的见解作为广告服务器的附加信号（在足球比赛中展示运动鞋广告比在游泳比赛中展示更具针对性），从而提供更相关的广告。

- 用户参与度 - 可以通过视频见解来提高用户参与度，方法是为用户提供相关的视频瞬间。 以某个教育视频为例，该视频在头 30 分钟讲述球体，在接下来的 30 分钟讲述锥体。 如果某个学生想要了解锥体，则该视频从 30 分钟处开始播放更合适。

有关详细信息，请参阅此[博客](http://aka.ms/videoindexerblog)。
