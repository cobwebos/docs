# <a name="entity-search-endpoints"></a>实体搜索终结点
实体搜索 API 包括一个终结点。

## <a name="endpoint"></a>终结点
要请求实体搜索结果，请将请求发送到以下终结点。 使用标头和 URL 参数来定义更多规范。

终结点 `GET`： 
``` 
https://api.cognitive.microsoft.com/bing/v7.0/entities
```

需要以下 URL 参数：
- mkt。 产生结果的市场。 
- q。 实体搜索查询。
